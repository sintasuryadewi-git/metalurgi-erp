# Build Project

This workflow is used to install all dependencies and build the application.

1. Ensure you have the correct Node.js version installed (>= v20). If `nvm` is used, ensure it is loaded.
// turbo-all
2. Run `npm install` to install dependencies.
3. Run `npm run build` to build the Next.js application.
