# Preview Project

This workflow starts the application's preview dev server locally.

1. Ensure you have the correct Node.js version installed (>= v20). If `nvm` is used, ensure it is loaded.
// turbo-all
2. Run `npm run dev` to start the Next.js development server.
3. Use the `browser_subagent` tool to navigate to `http://localhost:3000` to verify it's working properly.
